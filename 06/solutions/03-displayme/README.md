# 03 Display trade derive

This is exercise is about implementing an procedural macro.

To implement your solution, you must copy the example directory into a new directory at the same
level with a name that identifies your self.

The directory that you copied contains a Cargo workspace, with two crates.

You must implement the macro in the `displayme` crate and then run the tests of the `assignment`
crate which are in `src/main.rs`.
